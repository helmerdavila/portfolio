# Instalando Bat en Ubuntu Windows WSL

## Descargando bat

Me migré a Windows y al instalar Ubuntu a través de WSL echaba de menos de algunas herramientas que poseo en mi MacBook actual, **BAT** estaba entre una de ellas. Entonces como apenas encontré un par de guías por internet acá trataré de dejarlo como un recordatorio y con pasos sencillos

Primero abre tu terminal de Windows (con Ubuntu) y busca cual es la [última versión en Github](https://github.com/sharkdp/bat/releases). En mi caso era la `0.22.1`

Después comprueba que tengas wget instalado. La manera más común de hacerlo es revisarlo con `wget --version`. Si no lo tienes utiliza el siguiente script para hacerlo:

```bash
sudo apt-get install wget
```

Ya puedes instalar Bat con el siguiente script, recuerda reemplazar `VERSION` por la versión actual que viste en Github.

```bash
wget https://github.com/sharkdp/bat/releases/download/v${VERSION}/bat_${VERSION}_amd64.deb
# Si usamos por ejemplo 0.22.1
# wget https://github.com/sharkdp/bat/releases/download/v0.22.1/bat_0.22.1_amd64.deb
```

## Instalándolo con dpkg

Puedes usar dpkg para instalar paquetes Debian

```bash
sudo dpkg -i bat_0.22.1_amd64.deb
```
